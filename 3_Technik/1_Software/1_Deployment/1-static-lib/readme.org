#+TITLE: Readme

* Ordnerstruktur

/v1/ ist der Matlab Algorithmus hinterlegt, der im /X-Window/ Modus (Grafische
Oberflaeche OS) mit /Open-CV/ die Aufnahmen mit den dazugehoerigen /Bounding
Boxes/ ausgibt.

/v1_1/ ist der Algorithmus, der auf dem Jetson Nano installiert ist. Es werden
nur die Werte fuer /[Hoehe Breite PositionX PositionY Score]/ im Binaerformat
ueber /Stream/ ausgegeben.
