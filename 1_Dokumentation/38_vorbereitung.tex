\section{Vorbereitung}\label{vorbereitung}}

Zu Beginn als Vorbedingungen waren einige Punkte zur Überarbeitung des
Projektes angesetzt. Aufgrund dessen mussten
Vorarbeiten/Recherchearbeiten geleistet werden. Offene Fragen waren:
Integration von zusätzlicher Hardware zur Beschleunigung der
Echtzeitobjekterkennung, konkrete Umsetzung des Platinen-Redesignes,
Abänderung der Kamera zur Objekterkennung. Hierführ wurde eine
zweiwöchige Recherche durchgeführt. Kriterien waren:

\begin{itemize}
\item
  Kosten
\item
  Verfügbarkeit
\item
  Integration ins bestehende System
\item
  Leistung nach Integration
\item
  Vergleich der Maßnahmen
\item
  Aufwands-Nutzen-Schätzung
\end{itemize}
